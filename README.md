# Notes Collector

application can be used to

- save
- edit
- search

**Notes** with a lot of editing tools with the ability to save them offline on your device using the **PWA** feature

## Running Locally

```bash
$ git clone https://github.com/ahmedatefdev/notes-collector.git
$ cd notes-collector
$ yarn
$ yarn start
```
